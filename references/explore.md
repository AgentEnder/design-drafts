## Premise

Appeal to both ends of the ambition spectrum: promoters who may never want to be the biggest, *and* those who do. Showcasing a promotion going from nothing to huge would be awesome — but it must not intimidate new promoters. This is a **simple** marketing site for the admin-v2 product; theming should be done well but not be overbearing.

Built-in tension to resolve on purpose: **aspirational** (nothing → huge) vs **accessible** (don't scare newcomers).

### structure

All four below are `[picked]` — the user wants a mockup of each. Note: theme is held constant (House Lights Up), structure is the swept dimension, so the four drafts stay comparable (same skin, different skeleton).

- **Growth Ladder Scroll** [picked] — the page *is* the journey; scroll grows the promotion from empty VFW-hall card → packed arena, with House-Lights escalation (canvas brightens, crowd silhouette fills, headlines get bolder) driven by scroll position. Literal "nothing → huge."
  - keywords: scroll-narrative, escalation, journey, one-story
  - pushes against: #17
  - note: must degrade gracefully — `prefers-reduced-motion` + no-JS still give a complete page. Aim for the *cheap-and-tasteful* end (CSS scroll-driven animation / one IntersectionObserver), not pinned scroll-jacking.
- **Classic Stack, Confident** [picked] — honest fast marketing skeleton (hero / what-it-does / proof / waitlist), carried by strong type and restraint rather than novelty. The "simple and not overbearing" bet.
  - keywords: conventional, fast, conversion-clear, craft-over-trick
  - pushes against: it *is* close to #17 — the wager is execution over reinvention
- **Show Rundown** [picked] — page built like a match card: opener (hero) → mid-card (features) → main event (big-promotion payoff + CTA). Wrestling-native IA.
  - keywords: match-card, rundown, main-event-CTA, native-metaphor
  - pushes against: #3, #6 (eyebrow-pill hero)
- **Two-Track (Start Small / Go Big)** [picked] — explicitly serves both audiences via a light toggle/split: a calm "run your first show" lane and an aspirational "build a dynasty" lane.
  - keywords: dual-audience, toggle, split-path
  - pushes against: #17

### voice

- **Promoter-to-Promoter** [picked] — body register. Peer advice from someone who's run shows; warm and reassuring to newcomers, still respects ambition. e.g. "You've got a venue and eight wrestlers. We'll handle the paperwork so you can book."
  - keywords: peer, warm, reassuring, been-there
  - pushes against: #11 ("built for modern teams")
- **Ring Announcer** [picked] — headline register only (layered on top of the body voice). Big declarative poster-caps energy, e.g. "RUN. THE. SHOW." Welcomes via body copy, escalates via headline moments.
  - keywords: hype, declarative, all-caps, poster
  - pushes against: #15 ("we believe…")
- **Backstage Plain-talk** — direct, dry, concrete nouns over adjectives. Not picked, kept as negative space.
  - keywords: insider-plain, terse, concrete, unfussy

  - keywords: insider, playful, wink, lingo-forward
