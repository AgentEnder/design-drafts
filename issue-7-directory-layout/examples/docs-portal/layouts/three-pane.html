tinderbox layout: three-pane

Three-pane reference layout

Left rail for the doc tree, right rail for the current page outline, content in the middle. The reference style.

spark

tinderbox spark <name> [--ttl=<days>]

Creates a new branch and stamps a TTL.

list

tinderbox list [--json]

Lists known sparks and their remaining TTL.

burn

tinderbox burn [--remote] [--force]

Deletes expired sparks.

Exit codes

0 on success, 1 on generic failure, 2 on git-level failure, 3 if burn refused.
