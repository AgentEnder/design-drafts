tinderbox theme: console

Console mode

The console theme leans into the fact that this is a CLI tool. The body face is mono, the palette is terminal-adjacent, and headings get less weight because the type already commands attention by being mono.

This theme is best on the API reference page, where most of the content is signatures and param tables. It is worse for long prose — the mono body face is hostile to reading three paragraphs in a row.

$ tinderbox list
spark/fix-pricing-typo   13d remaining
spark/audit-log-spike     2d remaining

Notice that the in-page code tags blend into the body more here than in the paper theme. That's deliberate — when everything is mono, code stops being visually special.
