tinderbox api · paper · single-column

API reference

Three commands. All flags are listed below; each command also prints --help.

spark

tinderbox spark <name> [--ttl=<days>] [--from=<ref>]

Creates a new branch and stamps a TTL into .tinderbox/branches.json.

<name>
Slug appended to the spark/ prefix.
--ttl
Days until the branch is eligible for burn. Default 14.
--from
Base ref. Default HEAD.

list

tinderbox list [--json] [--expired-only]

Lists known sparks and their remaining TTL.

burn

tinderbox burn [--remote] [--force] [--dry-run]

Deletes expired sparks. Refuses branches with unpushed commits unless --force.

Exit codes

0 on success, 1 on generic failure, 2 on git-level failure, 3 if burn refused.

JSON format

{ "name": "spark/fix-pricing-typo",
  "createdAt": "2026-05-08T14:22:00Z",
  "ttlDays": 14,
  "expiresAt": "2026-05-22T14:22:00Z",
  "expired": false }
