Quickstart

Five minutes from install to your first reaped branch.

1. Install

brew install tinderbox
# or: curl -sSL https://tinderbox.dev/install.sh | sh

2. Spark a branch

Inside any git repo:

$ tinderbox spark fix-pricing-typo
created branch spark/fix-pricing-typo
TTL: 14 days (expires 2026-05-22)

spark creates a branch from the current HEAD and stamps a TTL into .tinderbox/branches.json. Push it like any other branch.

3. List what's outstanding

$ tinderbox list
spark/fix-pricing-typo   13d remaining
spark/audit-log-spike     2d remaining  (push first!)
spark/old-experiment      expired

4. Burn the expired ones

$ tinderbox burn
deleting spark/old-experiment ... done
1 branch reaped, 2 remaining
Heads up: burn deletes both the local branch and (with --remote) the remote tracking branch. It refuses to delete branches that have unpushed commits unless you pass --force.
