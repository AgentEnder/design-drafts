Quickstart — try each step

Run each block, then read the explanation below it.

1. Install

$ shelledit and re-run
==> Installing tinderbox Pouring tinderbox-0.4.bottle.tar.gz ==> Successfully installed tinderbox

Or: curl -sSL https://tinderbox.dev/install.sh | sh.

2. Spark a branch

$ tinderboxedit and re-run
created branch spark/fix-pricing-typo TTL: 14 days (expires 2026-05-22)

spark creates a branch from the current HEAD and stamps a TTL.

3. List what's outstanding

$ tinderboxedit and re-run
spark/fix-pricing-typo 13d remaining spark/audit-log-spike 2d remaining (push first!) spark/old-experiment expired

4. Burn the expired ones

$ tinderboxedit and re-run
deleting spark/old-experiment ... done 1 branch reaped, 2 remaining
